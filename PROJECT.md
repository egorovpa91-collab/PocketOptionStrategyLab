## Indicator Engine

Engine рассчитывает индикаторы только по закрытым M1-свечам.

Реализованы:

- `ClosePriceIndicator`;
- универсальный `EMAIndicator`;
- EMA 5, 8, 10, 13, 20, 30, 50, 100 и 200;
- инициализация EMA через SMA первых `period` свечей;
- рекурсивный расчёт по всей доступной закрытой истории;
- проверка минимального количества свечей;
- единый снимок индикаторов для одной закрытой свечи;
- защита от повторного расчёта timestamp.

Engine получает всю доступную историю Candle Manager. Поле
`required_candles` используется только как минимальный порог готовности,
а не как ограничение длины входной последовательности.
