# Pocket Option Strategy Research Lab

## Назначение

Это **не торговый бот**. Проект представляет собой профессиональную
исследовательскую лабораторию торговых стратегий для Pocket Option.

Система собирает рыночные данные, хранит историю, рассчитывает индикаторы,
тестирует стратегии, ищет устойчивые закономерности и в будущем выдаёт
сигналы `CALL`, `PUT` или `NO SIGNAL` для 10-минутной экспирации.

Автоматическое открытие сделок не входит в цели проекта.

## Scanner

Scanner стабильно:

- запускает Brave;
- подключается к вкладке Pocket Option через CDP;
- находит Market WebSocket;
- получает OHLC и историю;
- переключает 6 активов через интерфейс Pocket Option;
- проверяет фактически открытый актив;
- фильтрует пакеты только по активной паре;
- формирует и сохраняет M1-свечи;
- выполняет UPSERT по ключу `asset + timeframe + timestamp`;
- различает `INSERTED`, `UPDATED`, `UNCHANGED`;
- публикует `new_closed_candle`;
- использует 10 секунд на актив, полный цикл занимает около 60 секунд;
- корректно останавливается без закрытия Brave.

Индикаторы не рассчитываются по тикам. Тики используются только для
формирования текущей свечи и обновления High, Low и Close.

## Candle Manager

Candle Manager:

- подписывается на `new_closed_candle`;
- хранит только закрытые свечи;
- ведёт отдельную историю для каждой пары и таймфрейма;
- при первом событии восстанавливает до 500 закрытых свечей из SQLite;
- не включает формирующуюся свечу;
- предоставляет неизменяемый снимок истории будущему Indicator Engine;
- сообщает, достаточно ли истории для конкретного индикатора;
- контролирует порядок timestamp и фиксирует разрывы.
