# CHANGELOG

## v0.3.1 — Цикл из 6 активов

- Список Scanner сокращён с 9 до 6 согласованных активов.
- Удалены `GBPJPY_otc`, `AUDCAD_otc` и `EURRUB_otc`.
- При интервале 10 секунд полный цикл теперь занимает около 60 секунд.
- Механизмы CDP, переключения, фильтрации, UPSERT и Candle Manager не изменялись.
- Обновлены PROJECT.md, ROADMAP.md, IDEAS.md и CHANGELOG.md.

## v0.3.0 — Candle Manager

- Добавлен модуль `candle_manager`.
- Добавлена подписка Candle Manager на `new_closed_candle`.
- Реализовано отдельное хранение закрытых свечей по `asset + timeframe`.
- Добавлено восстановление до 500 закрытых свечей из SQLite.
- Добавлен запрос `get_candles_until()`.
- Формирующаяся свеча исключена из истории Candle Manager.
- Добавлены методы получения снимка истории и последней закрытой свечи.
- Добавлена проверка достаточности истории для будущих индикаторов.
- Добавлена диагностика разрывов timestamp.
- Добавлена итоговая статистика Candle Manager.
- Обновлены PROJECT.md, ROADMAP.md и IDEAS.md.

## v0.2.0 — UPSERT и закрытие свечи

- Реализован UPSERT по ключу `asset + timeframe + timestamp`.
- Добавлены статусы `INSERTED`, `UPDATED`, `UNCHANGED`.
- Формирующаяся свеча обновляется при изменении OHLC.
- Реализовано определение закрытия M1-свечи.
- Добавлено событие `new_closed_candle`.
- Первый исторический пакет не создаёт ложных событий закрытия.
- Подтверждена отдельная обработка состояния каждой пары.

## v0.1.0 — Фундамент

- Создан новый проект.
- Создана структура каталогов.
- Настроено виртуальное окружение.
