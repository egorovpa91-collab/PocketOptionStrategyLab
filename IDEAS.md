# IDEAS

## EMA

- Использовать периоды 5, 8, 10, 13, 20, 30, 50, 100 и 200.
- Не добавлять все периоды подряд без статистического обоснования.
- Позже вычислять наклон EMA, расстояние цены до EMA и спреды между EMA.
- Отдельно исследовать число свечей после пересечения EMA.
- Проверять устойчивость комбинаций вне обучающей выборки.
- Не считать конкретный период заранее лучшим для 10-минутной экспирации.

## Indicator Engine

- Индикатор получает всю доступную закрытую историю.
- required_candles является минимальным порогом готовности.
- Один набор индикаторов переиспользуется тысячами стратегий.
- Добавить составные результаты для ADX, +DI и -DI.
- Добавить тесты на заранее рассчитанных последовательностях.
